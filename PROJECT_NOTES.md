# 數學速解尋寶記 — 專案交接說明（給任何 AI 模型／協作者）

> 用法：開新對話時，把這份文件＋最新的 index.html 一起貼給模型，再說明這次要改什麼。

## 一、專案是什麼
澎湖縣親子科學園遊會攤位遊戲：玩家在 180 秒內用「兩位數乘法速算口訣」破解 3 道密碼鎖、打開寶箱，由關主蓋章過關。單一 HTML 檔的 Web App（PWA），部署於 GitHub Pages：https://squidlaw0805.github.io ，7–10 台 iPad 以「加入主畫面」執行，離線可玩。

## 二、檔案清單（repo 根目錄）
- `index.html`　遊戲本體（HTML＋CSS＋JS 全在同一檔，目前 v3.4）
- `sw.js`　離線快取 Service Worker（**每次改版必須同步改第 4 行版本字串**，如 math-treasure-v3.5）
- `manifest.json`、`apple-touch-icon.png`、`icon-512.png`　PWA 安裝用，通常不動
- 另有非 repo 文件：活動設計表 v3（Word）、A3 海報（pptx）、關主 Q&A 卡（Word）

## 三、遊戲規格（改動前必讀）
1. **三招題型**（每題物件由 genHeadSame / genTeens / genEleven 產生）：
   - 秘技一「頭同尾補」：十位同、個位和=10。combine:"concat"，前段=頭×(頭+1)、後段=尾×尾補0。
   - 秘技二「十幾乘十幾」：11–19。combine:"add"，前段=(被乘數+乘數個位)×10、後段=尾×尾，第三步輸入相加結果。
   - 秘技三「兩位數×11」：combine:"eleven"，第一步填「中間=頭+尾」、第二步輸入進位處理後的完整答案。個位固定 1–9（避 0）。
2. **兩段式輸入是核心賣點**：從介面上保證玩家用速算法而非直式硬算——不可為了簡化而改回單格直接輸入。
3. **模式**：正式版 3 題（TOTAL_Q=3）；小尋寶者 kid 模式 2 題（×11 不進位＋秘技一或二擇一），成功畫面顯示版本標示，過關標準一致。
4. **教學頁**：每招卡片＝「先當偵探」探究題（inqAnswer/inqReveal/resetInquiry，換頁自動重置）→ 原理推導 → 口訣 → 動畫（ANIM_DATA 資料驅動，t1/t2/t3）。
5. **失敗結算**：逐題顯示正解與 solveLine 拆解——這是教學回饋，不可移除。
6. 其他不變式：單一 HTML 檔、可離線、無任何個資蒐集、無排行榜（已刻意移除）、計時用時間戳（勿改回 setInterval 累加）。

## 四、改版流程
1. 請模型「只輸出要修改的區段」，人工貼回或請它給完整檔案後 diff 比對。
2. 升版本：index.html 右下 `.ver` 的 v3.x ＋ sw.js 的快取字串，兩處一起改。
3. 上傳 GitHub 覆蓋 → iPad 連網重整兩次 → 實測三種題型各一輪（含 ×11 進位題）、小尋寶者一輪、失敗一次看逐題回饋。
4. 若改了規則文字，同步檢查：教學卡、小抄浮層（hintOverlay）、海報、設計表是否一致。

## 五、測試清單（每次上線前）
- [ ] 三招各答對一次能解鎖；最後一題答對瞬間不會被「時間到」誤判
- [ ] ×11 進位題（如 67×11）第二步輸入 737 能過、輸入 637 有「記得進位」提示
- [ ] 答錯只清錯的那段、提示對應口訣
- [ ] 時間到 → 逐題回饋顯示正常、可立即再挑戰
- [ ] 偵探題猜錯出現「直接揭曉」；回教學頁會重置
- [ ] 斷網重新整理仍可載入（PWA 快取生效）

## 六、待辦點子（尚未實作，依效益排序）
1. 成功憑證防冒用：當日隨機圖章（改程式頂端一個常數）＋流水序號
2. 高手模式：連過 3 題解鎖「直接輸入完整答案」作為榮譽機制（學生若嫌兩段式煩再做）
3. 本機當日統計（挑戰次數／成功率／平均耗時，隱藏頁顯示，供評審參觀）
4. ×11 進位題比例目前約 56%（自然機率），若現場太難可在 genEleven 加權調成約 1/3
